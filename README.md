# controle-de-funcionarios
Sistema desenvolvido em Python que realiza um controle de funcionários

Desenvolvi esse sistema como um desafio proposto enquanto estou aprendendo Python.
Nele é possível cadastrar, editar, excluir, listar e pesquisar funcionários
Cada funcionário tem um nome, uma id e um cargo

Também foi criada a função de salvar os dados dos funcionários em um arquivo .txt

Foi um projetinho de cunho didático, sinto orgulho e realização por ter feito, quebrado a cabeça com algumas coisas que ainda não sabia, mas no final deu certo e me sinto realizado.
